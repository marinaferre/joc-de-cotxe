# joc-de-cotxe
A l'assignatura de programació avançada de la carrera de matemàtiques a la UAB vaig fer un joc d'un cotxe que va per una carretera. Amb les fletxes de l'ordinador cal intentar seguir la carretera sense surtir-ne i com més ràpid arribis a la següent moneda més puntuació obtindràs.

Per poder jugar al joc cal executar el fitxer anomenat el_joc
